# 🏨 Staylio

Staylio, kullanıcıların şehir, tarih aralığı ve misafir sayısı bilgilerini girerek konaklama seçeneklerini görüntüleyebildiği modern bir otel arama uygulamasıdır.

Proje kapsamında otel verileri, hava durumu, döviz kurları, kripto para fiyatları ve akaryakıt bilgileri gibi çeşitli veriler **RapidAPI** üzerinden alınarak kullanıcıya tek ekranda sunulmaktadır.

Bu projenin temel amacı; farklı dış servislerden veri çekme, gelen verileri listelemedir..

---

## 🚀 Proje Özellikleri

* 🔍 Şehir, giriş tarihi, çıkış tarihi ve misafir sayısına göre otel arama
* 🌐 RapidAPI üzerinden canlı konaklama verileri çekme
* 🌦️ Şehre özel hava durumu bilgilerini görüntüleme
* ⛽ Güncel akaryakıt fiyatlarını listeleme
* 💱 Anlık döviz kurları takibi
* ₿ Kripto para fiyatlarını görüntüleme
* 🏨 Otel kartlarında fotoğraf, puan, konum ve fiyat bilgileri gösterimi
* 📄 Detay sayfasına yönlendirme desteği
* 🎨 Responsive ve kullanıcı dostu arayüz
